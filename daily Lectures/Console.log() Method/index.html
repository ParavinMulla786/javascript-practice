console.log() in JavaScript

The console.log() method is one of the most important methods in JavaScript. It is used to print messages in the browser console.

Syntax:

console.log(msg1, msg2, ..., msgN);

Open console using: Ctrl + Shift + J

Interview Questions

1. What is console.log()?

👉 It is used to print output in the console for debugging.

2. Where does console.log() output appear?

👉

  • Frontend → Browser console
  • Backend → Terminal
