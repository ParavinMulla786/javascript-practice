JavaScript Comments

JavaScript comments are used to explain the purpose of the code. Comments are not executed by the program and are only for developers to understand the code better.

Uses of Comments

  • Explaining code
  • Documentation
  • Debugging (temporarily disable code)

Types of Comments in JavaScript

  • Single-line comments
  • Multi-line comments

Single-line Comments

Single-line comments start with // and continue till the end of the line.

Multi-line Comments

Multi-line comments start with /* and end with */.

Interview Questions

1. What are comments in JavaScript?

👉 Comments are used to explain code and are not executed by the browser.

2. How many types of comments are there?

👉 Two types:

  • Single-line (//)
  • Multi-line (/* */)

3. Why are comments important?

👉 They improve readability, help debugging, and make code easier to understand.
