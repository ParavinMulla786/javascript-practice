JavaScript Notes

What is JavaScript?

JavaScript is a lightweight, interpreted and dynamic programming language used to create interactive and dynamic web pages. It is widely used for both client-side and server-side development.

  • Lightweight and fast
  • Single-threaded
  • Event-driven
  • Cross-platform
  • Dynamically typed

Features of JavaScript

  • Client-Side Scripting: Runs in browser
  • Versatile: Used for frontend & backend
  • Event-Driven: Handles user actions
  • Asynchronous: Uses Promises & async/await
  • Rich Ecosystem: React, Angular, Vue

Client-Side JavaScript

  • Controls browser and DOM
  • Handles user events
  • Used for validation and UI interaction

Server-Side JavaScript

  • Runs using Node.js
  • Handles database, APIs, files
  • Used for backend development

History of JavaScript

  • Created by Brendan Eich (1995)
  • Initially called Mocha → LiveScript → JavaScript
  • Standardized by ECMA (ECMAScript)
  • ES6 (2015) major update
  • Latest version: ES14 (2023)

Advantages of JavaScript

  • Less server interaction
  • Fast response
  • Interactive UI
  • Rich user experience

Limitations of JavaScript

  • No direct file access (security)
  • No multi-threading
  • Limited system-level access

Imperative vs Declarative

  • Imperative: Step-by-step instructions (for loop)
  • Declarative: Focus on result (reduce method)

Development Tools

  • VS Code
  • Chrome DevTools
  • Git & GitHub
  • Webpack

Conclusion

JavaScript is one of the most powerful and widely used programming languages for building modern web applications. It supports both frontend and backend development.
