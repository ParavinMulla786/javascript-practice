JavaScript Variables

JavaScript variables are used to store data that can be changed later. They are like containers that hold values.

Ways to Declare Variables

  • Without keyword (not recommended)
  • Using var
  • Using let
  • Using const

Note:

let and const were introduced in 2015 (ES6). Before that, only var was used.

Variable Initialization

Assigning a value to a variable is called initialization.

Interview Questions

1. What is a variable in JavaScript?

👉 A variable is a container used to store data.

2. How many ways to declare variables?

👉 Four ways: var, let, const, and without keyword.

3. What is initialization?

👉 Assigning value to a variable.

4. Is JavaScript typed or untyped?

👉 JavaScript is a dynamically typed (untyped) language.
